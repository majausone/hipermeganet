Auto Assign Files

The auto-assign feature helps organize and link video files with their database entries automatically.

File Naming Convention

  • Videos should be named as: LocalID-ChannelName.mp4
  • Thumbnails should match: LocalID-ChannelName.jpg/png

Process

  1. Scans the global video path directory
  2. Matches files based on LocalID and Channel name
  3. Creates new video entries if none exist
  4. Updates paths for existing entries

Upload Process

New Video Upload

  • Checks if YouTube ID exists
  • If no ID exists, creates new video on YouTube
  • Updates local database with new YouTube ID

Video Update

  • If YouTube ID exists, performs metadata update
  • Optional thumbnail update
  • Preserves existing YouTube statistics

Bulk Operations

Bulk operations allow managing multiple videos simultaneously:

  • Upload multiple videos
  • Update visibility settings
  • Set language preferences
  • Manage categories
  • Apply kids content settings

Filters & Sorting

The application maintains filter and sort preferences:

  • Filters persist between tab changes
  • Column sorting order is saved
  • Channel selections are remembered
  • Use "Delete Local Data" to reset preferences

API Keys & Quotas

YouTube API

  • Quota limits apply to YouTube operations
  • Quota increases available through Google Cloud Console
  • Not available for purchase - must request increase

GPT API

  • Pay-per-use model
  • Approximately $0.10 for 18 channel translations
  • No quota limitations

Data Management

Local Database

  • Stores video metadata
  • Maintains channel information
  • Contains API keys (unencrypted)

Deletion Process

When deleting videos:

  • Only removes local database entry
  • Does not affect YouTube content
  • Maintains file system content

Translations

The translation system uses GPT-4 to translate:

  • Video titles
  • Descriptions
  • Tags
  • Thumbnail text

Channel Management

Channel Creation

  • Manual language assignment required
  • Auto-assigns settings to new videos
  • Inherits account settings

Synchronization

  • Updates metadata from YouTube
  • Preserves local settings
  • Maintains video associations
